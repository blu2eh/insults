# 辱华事件备忘录—犯我中华者，虽远必诛！
此站收集记录辱华事件，中国人应该记住那些侮辱过我们的国家、企业、品牌！有些事不应该被忘记，用自己能做的方式去爱国！



### 提交事件说明

网站采用 hexo 静态博客，文章采用 markdown 编辑。只需要你按照文件` _posts/apa` 里面的格式，提交md文件和图片即可。

新增一个文件，在 `_posts` 里面即可，可以加上你自己的信息和引用等等。

欢迎大家提交，谢谢！为中华之崛起而提交辱华名单吧！
